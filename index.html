ONEMANSHYO

About

ONEMANSHYO is an electronic music performance combining DJ mixing and live reactive visual mixing. Think of it as a one-person audio/visual show — live DJ set with real-time generated visuals that react to the music.

The custom app runs entirely in Chrome using WebGPU for shader rendering, Web Audio API for analysis and mixing, and the File System Access API for persistent library management. Zero installation. Single HTML file.

App Features

CREATE — Author WGSL fragment shaders with real-time preview, audio-reactive modulators (bass/mid/treble), FFT analysis, transform controls, and BumpR StickRs video overlays.
ORGANIZE — Library management with tagging, smart sets, search, and metadata editing. Containers package shader + audio + settings into portable .oms files.
PERFORM — 4-track mixer with 8 scenes, per-track EQ3, DJ FX, VFX, cue points, DSP compression, and quantized launching. Output composites to a pop-out window for live projection.
ADMIN — Audio card routing, camera/PIP configuration, BumpR StickRs management, effect pool assignments.

Technical

Optimized for Chrome on MacBook Pro 14". Uses WebGPU for 60fps shader rendering across 4 simultaneous tracks, with real-time audio analysis driving visual reactivity through bandpass-filtered peak detection with ADSR envelopes.

Yomatics

About

Yomatics explores cymatics — making sound vibrations visible. Water and materials react to electronic dance music frequencies in real-time, creating visual patterns that reveal the physical nature of sound.

Different frequencies create different geometric patterns. Bass frequencies produce large, slow-moving waves. Higher frequencies create intricate, detailed structures. The result is a direct, physical visualization of music.
